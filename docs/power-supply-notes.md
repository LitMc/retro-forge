# 電源供給メモ — 1998年のPCはこんなにシンプルだった

## 電源ケーブルの接続一覧

retro-forge（PII 400MHz + P3B-F + G200）の電源接続は**たった3本**。

| 接続先 | コネクタ | 備考 |
|--------|---------|------|
| マザーボード (P3B-F) | 20ピン ATX | これ1本でCPU含む全マザーボード回路に給電 |
| 光学ドライブ (PX-800A) | ペリフェラル4ピン | IDE機器の標準電源 |
| FDD (D353M D-5055) | FDD小型4ピン | Seasonic PRIME TX 750Wに標準付属 |

CF-IDEアダプタはIDEコネクタから給電されるため、電源ケーブル不要。

## CPUもGPUも電源ケーブルがいらない

現代のPC構築に慣れていると「CPUやGPUに電源を挿さなくていいのか」と驚く。

### CPU — 補助電源コネクタの歴史

- **Slot 1時代（1997-1999）**: マザーボードの20ピンATXからすべて給電。Pentium IIのTDPは24.6W（400MHz）で、20ピンで十分まかなえた
- **Socket 423/478（2000〜）**: Pentium 4で消費電力が急増。ATX12V 4ピン（通称「田の字」）が追加された
- **現代**: CPU補助電源は8ピン、さらにハイエンドでは8+4ピンや8+8ピンへ

### GPU — 補助電源コネクタの歴史

- **AGP初期（1997-2002）**: AGPスロットから最大25Wまで給電。Matrox G200やRiva TNTクラスは余裕で収まった
- **AGP 3.0/8x（2002〜）**: スロットからの給電上限が引き上げられたが、GeForce FX 5800系（2003年）で初めてペリフェラル4ピン補助電源が登場
- **PCIe時代（2004〜）**: 6ピン補助電源が標準化。現代のハイエンドGPUは12+4ピンで600W超を要求

### サウンドカード

PCI機器は全てスロットから給電。SB Live!（CT4830）も電源ケーブル不要。唯一の例外はCD-ROMからのアナログオーディオ入力用4ピンケーブルだが、これは電源ではなく信号線。

## 消費電力の比較

| 構成 | 総消費電力 | 電源ケーブル本数 |
|------|----------|----------------|
| **retro-forge (1998年構成)** | **約50W** | **3本** |
| 現代ミドルレンジPC (2026年) | 300-500W | 8-12本 |
| 現代ハイエンドPC (2026年) | 800W+ | 12本以上 |

Seasonic PRIME TX 750Wは明らかにオーバースペックだが、低負荷時にファンレス動作するため、むしろ静音メリットがある。
