# PS/2キーボード選定調査 (cmd_312)

## 1. NMB RT6652TWJP / RT6656TWJP

> **型番訂正**: 現在メルカリ/ヤフオクで出品されているNMBキーボードはRT6656TWJPであり、RT6652TWJPではない。

### スイッチ機構

- **方式**: NMB dome with slider（ラバードーム＋スライダー）
- メンブレン方式だが、スライダー付きの独自構造でメカニカルに近い打鍵感を実現
- NMBミネベア（日本精工系列）がOEM供給した業務用キーボードの定番スイッチ
- キーキャップは独自マウント（Cherry MX互換ではない）

### RT6652 vs RT6656の違い

| 項目 | RT6652TWJP | RT6656TWJP |
|------|-----------|-----------|
| 押下圧 | **重め** | **軽め**（RT6652の軽量版） |
| タクタイル感 | 明確 | やや軽いがタクタイル感あり |
| 評価 | 「重すぎる」という評価あり | RealForceに近い打鍵感と評価される |
| 打鍵音 | やや大きい | 比較的静か |

### 評価・評判

- 日本の自作PC界で1990年代に絶大な支持を受けた
- 「安くて打鍵感が良い」が代名詞
- OEM供給が多く、NEC/Epson/富士通ブランドでも流通
- RT6656TWJPは「ほぼRealForceの打鍵感」と評価するユーザーあり

### 価格

- **RT6656TWJP（現在出品中）**: 送料込み ¥7,230〜¥8,230
- **RT6652TWJP（過去落札相場）**: 約 ¥4,500
- かつてはヤフオクで¥100〜500で入手できたが、近年は希少化・価格上昇傾向

## 4. ダイヤモンド型方向キーとWin98ゲーム

### ★最重要調査結果★

Natural Keyboard Eliteの方向キーは**ダイヤ��ンド型（十字型・＋型）配列**であり、標準的な逆T字配列ではない。キーサイズも通常より小さい。

### 方向キー配列の問題点

| 問題 | 詳細 |
|------|------|
| 筋肉記憶との衝突 | 「下キーを押すつもりで右キーを押してしまう」報告多数 |
| 学習コスト | 分割キーボード自体より慣れるのが難しいとの評価 |
| キーサイズ | 通常キーより小さく、正確な入力が困難 |
| Insert誤爆 | 隣接するInsertキーを誤打する問題 |

Microsoftの後継モデル（Natural Ergonomic Keyboard 4000）では**逆T字配列に戻された**。設計者自身がダイヤモンド配列を「ミス」と認め、謝罪している。

### Win98時代のゲーム操作体系

#### 1997-1998年の操作方法

| 時期 | 主流の操作 | 備�� |
|------|-----------|------|
| ～1996年 | テンキー（2468）/ カーソルキー | 右手で移動操作 |
| 1997年 | 方向キーが主流、WASD台頭中 | Thresh（Quake大会優勝者）がWASD普及に貢献 |
| 1998年 | **WASD＋マウスが急速に標準化** | Half-Life、Starsiege: TribesがWASDデフォルト採用 |
| 1999年～ | WASD完全定着 | ほぼ全てのFPS/RPGがWASDデフォルト |

**結論**: 殿の対象ゲーム（MTG 1997年、MM6 1998年）はWASD普及の過渡期に発売。**方向キーがデフォルト**だが、MM6はGrayFace PatchでWASD化可能。

### Might and Magic VI の操作

| 操作 | デフォルトキー |
|------|---------------|
| 前進/後退 | **↑/↓ 方向キー** |
| 左右旋回 | **←/→ 方向キー** |
| 左右ストレイフ | Ctrl + ←/→ |
| 走る | Shift + 方向キー |
| 攻撃 | A |
| 呪文 | S |

- **方向キーが基本操作に必須**
- GrayFace Patch適用で`MouseLook=1`, `AlwaysStrafe=1`設定 → WASD+マウスルック可能
- テンキーでの移動も可能（Num Lock OFF時）

### MTG Shandalar の操作

| 場面 | 操作方法 |
|------|----------|
| デュエル（カード対戦） | **マウスのみ**（キーボード不要） |
| フィールド移動 | **方向キー**（上下左右の4方向） |
| ダンジョン/城 | **テンキー**（7,9,1,3で斜め移動も可能） |
| ワールドマジック | 1-5キー（ホットキー） |

- フィールド移動で方向キーを使用するが、**4方向のみ**で精密操作は不要
- ダンジョンでは**テンキーが推奨**操作（8方向移動対応）
- デュエル（ゲームプレイの大半）は**完全にマウス操作**

### ダ���ヤモンド型で困るゲームジャンル・場面

| ジャンル | 影響度 | 理由 |
|----------|--------|------|
| FPS（Quake等） | **高** | 方向キーでの高速移動＋周辺キーとの連携が必須 |
| アクションRPG | **中** | 方向キー連打・同時押しが頻繁 |
| CRPG（MM6等） | **低〜中** | 方向キー使用するが、高速操作は不要 |
| カードゲーム（MTG等） | **ほぼ無** | マウス中心、方向キーはフィールド移動のみ |
| シミュレーション | **低** | 方向キー使用頻度が低い |

### 結論: Eliteのダイヤ��ンド型は殿の用途で問題ないか

**条件付きYES（問題は軽微）**

- **MTG Shandalar**: デュエルはマウスのみ。フィールド移動で方向キーを使うが4方向・低速で問題なし。ダンジョンはテンキー推奨なのでダイヤモンド型は無関係。**→ 問題なし**
- **MM6**: デフォルトでは方向キー移動だが���GrayFace Patch適用でWASD化すればダイヤモンド型は無関係に。WASD化しない場合でも、MM6は高速精密操作が不要なため**影響は軽微**。**→ ほぼ問題なし（GrayFace Patch推奨）**
- **注意**: 将来FPSやアクションゲームをプレイする場合は問題が顕在化する可能性あり

### 「動作未確認品」のリスク評価

- PS/2キーボードは構造が極めて単純（ICチップ＋メンブレン/スイッチ＋コネクタ）
- 故障原因の大半: ケーブル断線、コネクタ接触不良（修理可能）
- スイッチ自体の故障は稀（特にラバードーム）
- **動作未確認 ≠ ��作不良**。多くは「確認環境がない」だけ
- リスク: 推定10-15%程度（PS/2ポートのないPCが増え、確認できない出品者が増加）
