# ケース調査 (cmd_301)

## RM42-502 互換性

### 公式スペック

| 項目 | 仕様 |
|------|------|
| フォームファクタ | 4Uラックマウントサーバーシャーシ |
| 対応マザーボード | SSI-EEB, SSI-CEB, E-ATX, ATX, Micro-ATX, Mini-ITX |
| 外部5.25インチベイ | **2基** |
| 外部3.5インチベイ（FDD用） | **0基（なし）** |
| 内部2.5インチベイ | 1基 |
| 寸法 | 430mm(W) × 176mm(H) × 468mm(D) |
| 拡張スロット | 7 |
| 電源 | 非搭載（ATX/PS2対応） |

### 搭載可否判定

- **5インチIDE CDドライブ**: ✅ 搭載可能（5.25ベイ×2）
- **3.5インチFDD**: ❌ 直接搭載不可（3.5外部ベイなし）。3.5→5.25変換ブラケットで搭載可能だが、CDドライブと共存すると空きベイ0
- **ASUS P2B/P3B-F（標準ATX）**: ✅ 搭載可能

### 結論: 非推奨

4Uラックマウント筐体であり、1998年代の自作PCタワーケースとは外観が全く異なる。CDとFDDの同時搭載も困難。

### 440BX Slot 1構成との互換性判定

#### 1. マザーボード搭載: ASUS P2B / P3B-F

| 項目 | 判定 | 詳細 |
|------|------|------|
| フォームファクタ | ✅ 搭載可能 | P2B/P3B-Fは標準ATX（305mm × 244mm）。FLP02はATX対応 |
| ネジ穴 | ✅ 互換 | ATXネジ穴規格は1995年以降不変。FLP02の標準ATXスペーサー位置で搭載可 |
| I/Oシールド | ⚠️ 要確認 | P2B/P3B-Fは当時のI/Oレイアウト（PS/2×2、シリアル×2、パラレル×1等）。FLP02付属I/Oシールドは現代規格のため、マザーボード付属品またはユニバーサルI/Oシールドが必要 |

**実績**: Hardware Asylumのレビューにて、Tyan Tiger 100 S1832D（Dual Slot 1 ATXマザーボード）がFLP02に問題なく搭載された実例あり。「for retro hardware in a modern chassis I am really impressed with how easy everything went together」とのコメント。

出典: [Hardware Asylum - Retro PC System Install](https://www.hardwareasylum.com/reviews/cases/sst_flp02/page5.aspx)

#### 2. Slot 1 CPU高さクリアランス

| 項目 | 寸法 | 判定 |
|------|------|------|
| FLP02 CPUクーラー高さ制限 | **182mm** | — |
| SECC/SECC2カートリッジ高さ（マザーボード面から） | 約127〜148mm（ヒートシンク含まず） | — |
| Intel純正ヒートシンク+ファン厚さ | 約25〜35mm | — |
| **合計推定高さ** | **約155〜165mm** | ✅ **余裕あり（17〜27mmマージン）** |

- SECC（初期Pentium II）: プラスチック+金属筐体、片面にアルミ放熱板。ヒートシンクは筐体前面に装着
- SECC2（後期PII / PIII）: 片面カバーのみ、チップ裏面にヒートシンク直接装着。やや薄型
- リテンションキット: マザーボード上のSlot 1コネクタ両脇にネジ留め。基板面からの追加高さは約5〜10mm
- **結論**: 182mmの高さ制限に対し、Slot 1 CPU + 純正ヒートシンクは約155〜165mm。**十分なクリアランスが確保される**

注: 寸法はIntel公式データシートの参考値および実測ベースの概算。正確な値はIntel Pentium II/III Datasheet (Order Number 243335/243502) を参照。

#### 3. AGPカード（3dfx Voodoo3等）クリアランス

| 項目 | 寸法 | 判定 |
|------|------|------|
| FLP02 GPU長さ制限（ドライブケージ有り） | **279mm** | — |
| FLP02 GPU長さ制限（ドライブケージ無し） | **386mm** | — |
| Voodoo3 3000 AGPカード長さ | 約170〜180mm（ハーフレングス） | ✅ **余裕あり** |
| Creative SB Live! CT4830（PCI） | 約170mm（ハーフレングス） | ✅ **余裕あり** |

- 当時のAGPカード（Voodoo3、TNT2、GeForce 256等）はいずれもハーフレングス（約170〜220mm）
- FLP02のドライブケージを装着した状態でも279mmあり、当時のAGP/PCIカードは全て収まる
- フルレングスカード（約312mm）も279mmのケージ有り状態では入らないが、ケージを外せば386mmまで対応

#### 4. 5.25インチベイ活用（CD-ROM + FDD）

FLP02は5.25インチベイ × 3基を装備。以下の構成が可能:

| ベイ | 搭載デバイス | 方法 |
|------|-------------|------|
| 5.25ベイ #1 | IDE ATAPI CD-ROMドライブ | 直接搭載 |
| 5.25ベイ #2 | 3.5インチFDD | 3.5→5.25変換ブラケットで搭載 |
| 5.25ベイ #3 | 空き（予備） | — |

**3.5→5.25変換ブラケット**:
- **StarTech BRACKETFDBK**: 3.5インチデバイスを5.25インチベイに取付ける定番ブラケット。黒ベゼル付き
  - Amazon.com: 約$8〜15
  - Amazon.co.jp: 約¥1,000〜2,000
- **3Dプリント**: Thingiverseに無料デザイン多数あり（thing:1279411等）

**注意**: FLP02のFDDスロット（前面最下段）はダミー装飾であり、実際のドライブベイではない。FDD搭載は必ず5.25ベイ+変換ブラケットで行う。

Hardware Asylumのレビューでは、Teac FD-55GFR（5.25インチフロッピードライブ）を5.25ベイに直接搭載した実例もあるが、「SilverStoneのデザインは『もう少し』のところ」とのコメントがあり、ベイカバーとの完全なフィットには若干の調整が必要な場合がある。

#### 5. 電源互換性（ATX 24pin ↔ 20pin）

| 項目 | 仕様 | 判定 |
|------|------|------|
| Seasonic PRIME TX 850W | ATX 24pin（20+4pin分離式）、奥行き**170mm** | — |
| FLP02 PSU奥行き制限 | **250mm** | ✅ **80mmマージン** |
| 440BX（PIIX4E）マザーボード | ATX **20pin** | — |
| 互換性 | 20+4pin分離式 → 20pin部分のみ接続 | ✅ **互換** |

- Seasonic PRIME TX 850Wの24pinコネクタは**20+4pin分離式**。20pin部分のみをP2B/P3B-Fの20pinヘッダに接続し、4pin部分は未接続のまま浮かせる
- 電気的に完全互換。追加の3.3Vラインは440BXでは不要
- 850Wは440BX構成（実消費100W未満）に対して大幅なオーバースペックだが、動作に問題なし
- **注意**: 出力が低すぎると逆に問題になるケースがあるが、750Wクラスは低負荷時でも安定動作する

#### 6. 拡張スロット構成

FLP02は7+2スロット（7標準 + 2垂直）。440BX構成での使用:

| スロット | 用途 |
|---------|------|
| AGP × 1 | 3dfx Voodoo3 3000 |
| PCI × 1 | Creative SB Live! CT4830 |
| PCI × 4 | 空き（ネットワークカード、SCSI等に使用可能） |

P2B/P3B-FはAGP×1 + PCI×5 + ISA×1（P2B）/ ISA×0（P3B-F rev1.04）。FLP02の7スロットで全て収まる。

### 総合判定

| 項目 | 判定 | 備考 |
|------|------|------|
| ATXマザーボード（P2B/P3B-F） | ✅ | 標準ATXネジ穴で搭載可能。実績あり |
| Slot 1 CPU高さ | ✅ | 182mm制限に対し約155〜165mm。余裕あり |
| AGP/PCIカード | ✅ | 当時のカードはハーフレングス。279mmで余裕 |
| CD-ROM + FDD | ✅ | 5.25ベイ3基で両方搭載+予備1基 |
| 電源 | ✅ | 20+4pin分離で20pin互換。奥行きも余裕 |
| I/Oシールド | ⚠️ | マザーボード付属品またはユニバーサル品が必要 |

**結論: FLP02は440BX Slot 1構成と完全に互換性あり。** レトロ外観・新品入手・十分なベイ数を兼ね備え、このビルドに最適なケースである。唯一の注意点はI/Oシールドの用意とFDD搭載時の変換ブラケットの必要性のみ。
