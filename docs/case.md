# ケース調査 (cmd_301)

## RM42-502 互換性

### 公式スペック

| 項目 | 仕様 |
|------|------|
| フォームファクタ | 4Uラックマウントサーバーシャーシ |
| 対応マザーボード | SSI-EEB, SSI-CEB, E-ATX, ATX, Micro-ATX, Mini-ITX |
| 外部5.25インチベイ | **2基** |
| 外部3.5インチベイ（FDD用） | **0基（なし）** |
| 内部2.5インチベイ | 1基 |
| 寸法 | 430mm(W) × 176mm(H) × 468mm(D) |
| 拡張スロット | 7 |
| 電源 | 非搭載（ATX/PS2対応） |

### 搭載可否判定

- **5インチIDE CDドライブ**: ✅ 搭載可能（5.25ベイ×2）
- **3.5インチFDD**: ❌ 直接搭載不可（3.5外部ベイなし）。3.5→5.25変換ブラケットで搭載可能だが、CDドライブと共存すると空きベイ0
- **ASUS P2B/P3B-F（標準ATX）**: ✅ 搭載可能

### 結論: 非推奨

4Uラックマウント筐体であり、1998年代の自作PCタワーケースとは外観が全く異なる。CDとFDDの同時搭載も困難。
