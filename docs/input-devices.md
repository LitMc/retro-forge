## 1998-2000年の定番機種

### キーボード

1. **IBM Model M Enhanced Keyboard (1391401)** — バックリングスプリング。伝説的打鍵感。重量2.0-2.5kg。鉄板バックプレート。1998-2000年当時も「最強キーボード」として認知。Unicomp後継が現行販売中。中古eBay $50-150
2. **Cherry G80-3000** — Cherry MXスイッチ。Gold Crosspoint接点で5000万回以上耐久。1990年代製PBTキャップ付きは特に評価が高い。現行販売継続中
3. **NMBミネベア RT6652TWJP** — 安くて打鍵感が良い。日本の自作PC界で絶大な支持。OEM供給多数。ヤフオクで100-500円。状態良品は希少
4. **Microsoft Natural Keyboard Elite (1998年)** — エルゴノミクス分割デザイン。長時間タイピングのプログラマーに支持。中古eBay $20-50
5. **Microsoft Natural Keyboard Pro (1999年)** — 上位モデル。USBハブ2ポート内蔵。ホットキー19個。中古eBay $30-80

### マウス

1. **Microsoft IntelliMouse Explorer (1999年10月)** — 世界初の一般向け光学式マウス。5ボタン。銀色ボディに赤テールライト。革命的製品。ヴィンテージ品 eBay $30-100
2. **Microsoft IntelliMouse Optical (2000年4月)** — Explorer左右対称版。5ボタン。後継1.1Aが長期販売
3. **Microsoft IntelliMouse ボール式 (1996年)** — スクロールホイール搭載マウスの元祖的存在。光学式登場前の大定番。eBay $10-30
4. **IBM ScrollPoint (1997年8月)** — ミニジョイスティック搭載（上下左右全方向スクロール）。ThinkPadトラックポイントに通じる操作感。eBay $15-40
5. **Logitech MouseMan Dual Optical (2001年)** — 2つの800dpiセンサーを45度配置。精度を求めるゲーマーに評価。eBay $20-50
6. **Logitech MouseMan Wheel (1999年)** — テクスチャ付きグリップ。4ボタン。右手専用人間工学形状。eBay $10-30
