## セクション1: 外してはいけないポイント（必須制約一覧）

### パーツ間の依存関係（「○○を買うなら××必須」）

| 買うもの | 必須条件 | 理由 |
|----------|----------|------|
| Slot 1 CPU (PII/PIII/Celeron) | → Slot 1対応マザーボード必須 | Socket 370ボードにはSlot 1 CPUは物理的に刺さらない |
| ASUS P2B/P2B-F/P3B-F | → Slot 1 CPU必須 | これらはSlot 1ボード |
| AGP GPU (G200等) | → AGPスロット付きMB必須 | P2B/P2B-F/P3B-FはAGPあり。問題なし |
| PC100 SDRAM | → DIMM対応MB必須 | P2B系はDIMM対応。問題なし |
| ATX電源 | → ATXフォームファクタMB必須 | P2B(ATX)/P2B-F(ATX)/P3B-F(ATX)は全てATX |
| Slot 1 CPU | → Slot 1用CPUクーラー必須 | PII 350MHz SL2U3はSECC1パッケージ。SECC1対応品を選ぶこと |

### ❌ 買ってはいけないもの（NGパターン）

| ❌ NG | 理由 | 見分け方 |
|--------|------|----------|
| ❌ Socket 370 マザーボード | Slot 1 CPUが使えない | ソケットの形状が全く異なる（ZIF vs カートリッジ） |
| ❌ PCI版 G200 | AGPスロットを使わない=帯域損失 | カード端子の切り欠き位置が異なる。PCI=長い端子1本、AGP=短い端子+切り欠き |
| ❌ PC66 / EDO SDRAM | 440BXはPC100必須（PC66は動作不安定）| メモリモジュールのラベルを確認。「PC100」or「PC133」の表記必須 |
| ❌ ASUS P2B-B | Baby ATフォームファクタ。ATXケースにネジ穴が合わない | 型番末尾「-B」。ATX電源コネクタも異なる |
| ❌ ASUS P2B-N | NLXフォームファクタ。専用ケース必要 | 型番末尾「-N」。ライザーカード構造で一般ケース不可 |
| ❌ AT電源 | ATXマザーに接続不可（コネクタ形状・ピン配置が異なる） | ATX=20pin or 24pin。AT=6pin×2（P8/P9）|
| ❌ Dell OEM版 SB0220等 | EMU10K1X搭載（劣化版チップ）。DirectSound HWアクセラレーション非対応 | 「SB0220」「Dell」の表記に注意。CT型番（CT4780/4830/4870）なら安全 |
| ❌ SECC2専用クーラー | PII 350MHz SL2U3はSECC1パッケージ。取付穴が異なる | クーラーの製品説明に「SECC2」のみ対応と書いてあれば不可 |
| ❌ ISA版サウンドカード + 現行ATX電源 | ISAカードは-5Vを必要とする場合あり。現行ATX電源は-5V非搭載 | YMF744はPCIカードなのでこの問題は発生しない |

### ⚠️ 注意が必要なもの

| ⚠️ 注意 | 対処法 |
|----------|--------|
| P3B-Fで-5Vエラーが出る | BIOS設定で解決: CMOS Setup → Power → Hardware Monitor → -5V → [Ignore] |
| P3B-Fのリテンション（CPU固定金具）破損 | 致命的ではない。結束バンドで応急対応可。eBayで交換品$10-20 |
| P3B-FにはISAスロットが1本のみ | ISAサウンドカードを使わない限り問題なし。SB Live!はPCI |
| P2B-FにはSCSIが内蔵 | 機能同等。SCSI不要でも問題なし（使わなければよい） |

## セクション2: 自由度のある部分（代替OK一覧）

### CPU: PII 266〜400MHz帯 または Celeron 300A

| CPU | MTG速度 | 安心度 | 備考 |
|-----|---------|--------|------|
| **PII 350MHz** ★推奨 | ◎ 実証域中央 | ◎ OC誘惑なし | 安心構成の本命 |
| PII 300MHz | ○ 実証域下限 | ◎ | 最安。速すぎる心配なし |
| PII 400MHz | ○ 実証域上限 | ○ | やや速い可能性あり。cpukiller3で調整可 |
| PII 266MHz | △ 実証域下限以下 | ○ | MTG動作報告ありだが余裕少ない |
| Celeron 300A | ◎ 300MHz定格 | △ OC誘惑あり | ロマン構成向き。安心構成ならPIIが無難 |
| Celeron 333/366/400MHz | ○ | ○ | MTG速度域内なら問題なし |

**要点**: MTGはCPUクロック依存でゲーム速度が変わる。PII 266-400MHzが実証済み安全域（VOGONS/AnandTech）。速すぎる場合はcpukiller3（Win98対応フリーウェア）で調整可能。

### MB: P2B / P2B-F / P3B-F のどれでも可

| マザーボード | 特徴 | ISA | DIMM | Coppermine |
|-------------|------|-----|------|------------|
| **P2B** | 440BXリファレンス完成形 | 3本 | 3本 | 非対応 |
| **P2B-F** | P2B + PCI×5 + DIMM×4（ISA 1本減） | 2本 | 4本 | 非対応 |
| **P3B-F** ★現実的 | P2B後継。BIOS充実。Coppermine対応 | 1本 | 4本 | 対応 |

- P2B無印ATX版は2026年現在ほぼ入手困難（cmd_262調査時点で出品ゼロ）
- P3B-Fが最も現実的な選択肢。P2Bと同一チップセット（440BX）で機能上位互換
- 全てATXフォームファクタ。SilverStone RM42-502に搭載可能

### GPU: G200の型番・リビジョン違いは問題なし

| 型番表記 | 正体 | 問題 |
|----------|------|------|
| **G2+/MILA/8** | Millennium G200 AGP SGRAM 8MB | なし |
| **798-02 REV_A** | 同上（基板番号表記） | なし |
| **G2+/MILA/8I** | 同上 | なし |
| **MGA-G200A-D** | 同上（チップ型番表記） | なし |
| **Mystique G200** (G200MMS) | SDRAM版。やや低速だが同一ドライバ | ○ 代替可 |

**要点**: 「Matrox G200」「AGP」「8MB」の3条件を満たせばどの型番でも可。PCI版のみ不可。

### サウンド: EMU10K1搭載なら型番不問

| 型番 | 名称 | 互換性 |
|------|------|--------|
| **CT4780** | SB Live! Value (リテール) | ★本命 |
| **CT4830** | SB Live! Value (OEM) | ★完全互換 |
| **CT4870** | SB Live! Value (改良版) | ★完全互換 |
| **CT4760** | SB Live! X-Gamer | ★完全互換（デジタル出力あり） |
| **SB0200** | SB Live! 5.1 | ★完全互換 |
| **SB0100** | SB Live! 5.1 (初期) | ★完全互換 |
| SB0220 | Dell OEM | ❌ EMU10K1X（劣化版）の場合あり。避ける |

**要点**: チップに「EMU10K1」と刻印されていれば安全。「EMU10K1X」は別物（劣化版）。

### 後から気づきがちな小物リスト

買い忘れ防止チェックリスト:

- [ ] **IDEケーブル** — マザーボードに付属していない場合が多い
- [ ] **CR2032電池** — 27年経過品は100%電池切れ。購入必須
- [ ] **VGAケーブル** — CRTモニタとの接続用。D-Sub 15pin
- [ ] **CPUクーラー** — PII 350MHzクーラーなし品を買った場合は別途必要
- [ ] **メモリ予備** — PC100 SDRAM は不良品率が高い。2枚以上確保
- [ ] **スペーサー（六角）** — ケースにマザーを固定する金属スペーサー。RM42-502に付属しているか確認
- [ ] **ネジ類** — インチネジ（HDD/光学ドライブ用）とミリネジ（FDD等）
- [ ] **PS/2キーボード** — Win98はUSBキーボード非対応の場合あり。PS/2推奨
- [ ] **PS/2マウス** — 同上。PS/2マウスを用意

### CF-IDE変換について

- **40pin (3.5インチ) 版を選ぶ**: P2B/P3B-FのプライマリIDEは40pinコネクタ
- **CFは8GBで十分**: Win98 SE + MTG + SimCity 2000 で2GB未満。余裕あり
- **IDEモード**: CF-IDE変換はTrue IDEモードで動作。Win98から普通のHDDとして認識される
- **利点**: HDDより無音・低発熱・故障率激低。レトロPC構築の定番手法
- **注意**: CFカードはFDISK→FORMATが必要（Win98インストーラで実行可能）

### ATX電源について

- **300Wで十分**: PII 350MHz構成の消費電力は100W以下
- **20+4pin対応品で問題なし**: P2B/P3B-Fは20pinコネクタ。24pin電源の4pin部分を外して使用
- **-5V非搭載でも問題なし**: P3B-Fの場合、BIOSで-5Vエラーを[Ignore]にするだけ（セクション1参照）
- **秋葉原で買える店**: ツクモ、ドスパラ、パソコン工房、じゃんぱら等で新品在庫あり

### SilverStone RM42-502 との相性

- **フォームファクタ**: 4U ATXラックケース → P2B/P2B-F/P3B-F（全てATX）は搭載可能
- **電源**: ATX電源搭載可能。300W新品で問題なし
- **Slot 1 CPU高さ**: SECC1カートリッジは高さ約130mm。4Uケース内部高約170mm → 収まる
- **AGP/PCIスロット**: 標準ATX配置。G200 AGP + SB Live! PCI は問題なく装着可能
- **ドライブベイ**: CF-IDE変換を3.5インチベイに固定（または両面テープでケース内設置）

## セクション5: 当日の判断基準

### 1. コンデンサの目視チェック方法

マザーボード購入時に最も重要な確認ポイント。1999-2003年製ボードはCapacitor Plague（コンデンサ不良問題）の影響時期に該当する。

#### 正常品の外観
- 天面（上面）が**完全に平ら**
- 側面に膨れ・染み・変色なし
- 基板上にコンデンサ由来の液体痕なし
- ラベル（メーカー名）が読める状態

#### 膨張の見方（天面のドーム型膨れ）
- コンデンサ天面の**十字刻印（防爆弁）が盛り上がっている** → NG
- 天面が**ドーム状に膨らんでいる** → NG（重度劣化）
- 側面が**太鼓状に膨らんでいる** → NG
- **1本でも膨張コンデンサがあればそのボードは避ける**（他も劣化進行中の可能性大）

#### 液漏れの見方（基板上の茶色い染み）
- コンデンサ根元（基板との接合部）に**茶色〜オレンジの染み** → 液漏れ。NG
- コンデンサ側面に**乾燥した液体の跡** → 液漏れ。NG
- 基板裏面（はんだ面）にコンデンサ直下の**染み** → 液漏れ。NG

#### ASUS P2B/P3B系のコンデンサ事情
- **日本製コンデンサ**（ニチコン Nichicon / ルビコン Rubycon / 三洋 Sanyo）搭載
- Capacitor Plague の原因は台湾メーカー製（Jackcon/Teapo/Luxon等）の不良電解液
- ASUS P2B/P3B系は日本製コンデンサのため**Plague耐性が高い**
- ただし27年経過 → 正常な経年劣化は避けられない。目視チェックは必須
- **見分け方**: コンデンサ側面のメーカーロゴ。「KZE」「KY」（ニチコン）、「YXA」「MCZ」（ルビコン）等

### 2. 動作未確認品を買う場合のリスク評価

#### 動作未確認でも安心なケース
- ✅ **老舗業者の出品**（評価1,000件以上・99%以上好評率）
- ✅ **詳細な商品説明**がある（型番・リビジョン・目視確認結果の記載）
- ✅ **コンデンサが目視で良好**（写真で天面平ら・液漏れなし確認済み）
- ✅ **PCパーツ専門店の店頭品**（秋葉原の有名ジャンクショップ）
- ✅ 価格が相場の半額以下（リスク込みの価格設定）

#### 動作未確認でリスク高いケース
- ⚠️ **個人出品・評価少ない**（評価10件未満）
- ⚠️ **商品説明がほぼない**（「動作未確認、ノークレームノーリターン」のみ）
- ⚠️ **コンデンサ状態が写真で確認できない**（写真1枚のみ・ぼやけている）
- ⚠️ **「水漏れで倉庫保管」**等の保管環境に不安
- ⚠️ **価格が相場と同等**（動作確認済み品と同じ価格なら確認済み品を選ぶ）

### 3. ジャンク品の心得

- **「ジャンク品」≠「壊れている」**: 多くは「動作確認する手間を省いた」品。7〜8割は正常動作する
- **「動作未確認」≠「動作しない」**: 出品者が確認環境を持っていないだけの場合が多い
- **保証なし = 自己責任**: 返品不可が基本。壊れていたら次を探す覚悟で
- **秋葉原店頭のメリット**: ヤフオクと違い実物を手に取って確認できる。コンデンサ目視は店頭が最強
- **複数候補を確保**: 特にマザーボードは予備を持つのが理想。メモリも2枚以上
- **交渉のコツ**: 秋葉原ジャンクショップは値引き交渉可の店も多い。まとめ買い割引を聞いてみる
